# maze solving

## building

```bash
make
```

## running

```bash
./limewater -h
```
```
Użycie: ./limewater PLIK_LABIRYNTU [OPCJE]
Program rozwiązuje labirynt w PLIK_LABIRYNTU.
Możliwe OPCJE:
  -b    interpretuj PLIK_LABIRYNTU jako plik binarny
```

example mazefiles can be found in the `data/` directory
